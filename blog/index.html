Blog 3: Sept. 19th, 2021

For this blog post we were asked to chose one of the aspects of web development and according to Forbes.com , there are 7 aspects of web development that are very important to attract users. For the purpose of this blog post, I am going to discuss the importance of navigation on a web page since we will be building a static site for a customer in this class.

Navigation is important in web developement because the users are looking for information and they don't want to hav to figure out how to work the website as well. For example, when I was looking for a source to use for this article, I ran into sites that made it hard for me to find the information I wanted. So of cource, in pure GenZ fashion, I just went to a different page to get the information. If was a user looking to buy an item for that site, the site would have lost my business. One way that can allow you site to be more user friendly in terms of navigation, is to have a floating optios or menu bar on the page.
An example of this would be Harrisonburgelectric.com's page. As you can see when you load the page, the menu bar is located at the top, but as you scroll down the menu bar stays at the top. This allows easy navigation to anywhere on the site.
